=== Kompliziert vs. Komplex

[cols="1a,.^1a"] 
|=== 
| 
==== Kompliziert
* das Maß an Ungewissheit
* Wissen fehlt
* Kompliziertes System
** zuverlässig, exakt
** von außen steuerbar
** konstantes Verhalten
| 
==== Komplex
* das Maß der Menge an Überaschungen
* Kontext-abhängig
* Komplexes System
** "lebendig", erzeugt Überaschungen
** von außen nur beobachtbar
** dynamisches Verhalten
|===
